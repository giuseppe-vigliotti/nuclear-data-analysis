# Nuclear Data Analysis

Exploratory data analysis on open nuclear/radiological datasets (IAEA PRIS).

## Stack
- Python (pandas, numpy, matplotlib, seaborn)
- SQLite + SQL
- Jupyter Notebook

## Project Structure
- `data/` — raw and processed datasets
- `db/` — SQLite database
- `notebooks/` — Jupyter EDA notebooks
- `sql/` — SQL queries
- `etl/` — data loading scripts
- `plots/` — exported charts
